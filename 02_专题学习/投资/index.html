一、看盘框架

每日看盘完整流程：从盘前预案到收盘复盘

盘前预案（每天必做）

1. 指数分析

  • 前一交易日收盘位置
  • 关键支撑/压力位（两道线）
  • 高开/低开/平开的不同含义
  • 量能预期

2. 情绪指标

选几个代表性标的作为风向标：

  • D头指标（多头代表）：看封单、溢价
  • K头指标（空头代表）：看是否修复

判断公式：D头强/弱 × K头强/弱 → 情绪偏暖/偏冷/弱平衡

3. 题材预判

  • 竞价最强方向是什么
  • 科技与老登（传统板块）的博弈预判
  • 核心票竞价金额排序

4. 个股预案

  • 持有个股的几种走势预案（强/中/弱）
  • 新开仓条件

竞价阶段（9:15-9:25）

核心判断链

指数位置 → 情绪强弱 → 题材方向 → 个股应对

开盘三问

  1. 指数开在什么位置？（支撑之上/之下/两道线之间）
  2. 涨跌家数如何？（决定今天是抱团还是普涨）
  3. 科技 vs 老登谁在动？（决定资金流向）

关键信号

  • 高开：高开分歧是卖点
  • 低开：低开利于消化分歧，若指数修复，科技有望修复
  • 证券拉升：判断是"助攻"还是"背刺"

开盘阶段（9:30-10:00）

科技开盘后的四种走法

走法应对
先修复偏减仓点，防回落
先分歧看水下低吸机会
直接走强持股，不封板减
直接走弱等修复，不割深水

老登与科技的博弈

老登拉升 + 证券背刺 → 科技减仓
老登投降 + 证券助攻 → 科技持股/加仓
老登拉升 + 量能够 → 可能高低切
老登高潮 → 科技可埋伏

盘中阶段（10:00-14:30）

10点早盘小结模板

  1. 指数：开盘位置 → 盘中走势 → 当前位置 → 今日定性
  2. 情绪：竞价情绪 → 盘中变化 → 涨跌停/涨跌比
  3. 题材：竞价方向 → 盘中轮动 → 当前主线
  4. 个股：持有个股的状态和操作建议

盘中关键信号

  • 涨跌家数变化 — 最重要的实时指标
  • 黄白线关系
  • 证券板块动向
  • 科技核心票动向
  • 新题材异动

题材运行节奏

这是复盘中反复出现的核心框架：

分歧 → 修复 → 高潮 → 兑现 → 分歧 → ...
阶段操作
分歧（第一天）核心票低吸，不追后排
修复持股，不封板减
高潮纪律减仓，不追高
三连高潮必须减仓，风险 > 机会
兑现/大分歧深水不割，等修复再减
判断口诀
  • "高潮一定是减仓点"
  • "分歧是低吸核心的机会"
  • "科技分歧一般不超过2天，第三天修复概率大"
  • "连续高潮次日冲高容易回落"

分歧深度判断：一冰 / 二冰 / 三冰

科技连续分歧时的节奏：

第一次分歧（一冰）→ 第一时间低吸核心
第二次分歧（二冰）→ 值得博弈，但偏左侧，控仓
第三次分歧（三冰）→ 概率极低，如果出现是重仓机会
核心原则
  • "科技分歧一般不超过2天，第三天修复概率大"
  • "今天低吸偏左侧，有不确定性；明天再分歧，就是不错的博弈点"
  • "科技三冰概率比较低"

左侧 vs 右侧交易

左侧右侧
特征提前埋伏，不等确认确认后再入场
优势赔率高，成本低确定性高
劣势可能继续跌成本高，利润空间小
适用二冰博弈、超跌票趋势确认、突破追入
"左侧有赔率但没有确定性，右侧有确定性但赔率就差一点"

情绪兑现 vs 逻辑兑现

大跌时必须区分：是逻辑坏了，还是情绪宣泄？

案例：7月2日 Meta 事件
Meta出租云算力 → 市场担心算力过剩 → 科技集体抛压

但逻辑站不住脚：
· AI Token价格持续走高
· 头部IDC算力租赁报价已回历史高位
· 下游大厂仍在加价抢算力
· Meta卖算力是因为自己大模型不行

结论：这是情绪兑现，不是逻辑兑现。
情绪兑现 → 深水是机会；逻辑兑现 → 必须离场。

收盘阶段

  • 今日操作复盘
  • 知识点小结
  • 明日预案初稿

看盘进阶：从数据到判断

信息处理的层次

  1. 看到：指数点位、涨跌家数、板块涨跌
  2. 判断：为什么涨/跌？谁在动？
  3. 应对：基于判断的操作策略
  4. 复盘：判断对不对？为什么？

经典判断链

涨跌家数 4000 家下跌 + 老登没起来 + 量能够
→ 资金没地方去
→ 只能回流科技
→ 科技深水是低吸机会

二、交易战法

11 个核心技法，从选股到风控

1. 炸板六原则

用于判断炸板后的回封概率：

1. 炸板时净额是否流入（不是流出）
2. 日K是否缩量（不是爆量）
3. 题材是否还在延续
4. 换手是否充分
5. 板块是否有支撑
6. 指数是否配合
经典案例：黄河旋风
炸板时净额流入 + 日K缩量 + 题材延续 → 良性分歧 → 回封

炸板补量要点

  • 一字板竞价尾声顶一字的 → 盘中分歧更大 → 等换手
  • "换手到前高 70% 以上（经验值）"→ 安全区间
  • "要么要确定性，要么要日内赔率"

2. 789 减仓法

高潮不封板的纪律性减仓：

科技高潮时，手里票 7%、8%、9% 又不封板 → 减仓
因为高潮都不封板，板块回落时它回落的概率大

20cm 票：翻倍，14%-18% 不封板就减

3. 买点后置

买点后置 = 等换手到前高 70% 以上
→ 确定性买点（回封确认后）
→ 日内赔率买点（深水低吸）

4. 利润垫策略

核心理念

  • 买入后第一时间做利润垫
  • 利润垫 = 减仓收回成本，留利润博弈
  • "利润垫够了就躺好"

操作步骤

  1. 买入后，拉升减仓 → 收回部分成本
  2. 剩余仓位全是利润 → 心态稳
  3. 5 日线不破就拿着
  4. 新高附近减仓做倒 T
利润垫厚度策略
< 10%谨慎，随时准备减
10% – 30%5 日线防守
30% – 50%可以格局，等新高
> 50%躺平，新高减即可

5. 高位看金额，低位看量

高位：只有绝对金额才能承接得住
低位：看量能变化，放量 + 止跌 = 底部信号

爆量模型

  • 爆量不跌 = 可能是吃货量
  • 爆量 + 反包 = 趋势预期
  • 爆量 + 下跌 = 出货

6. 涨跌家数判断法

这是最重要的盘中指标之一：

上涨家数 < 1000  → 极度冰点 → 资金抱团科技/主线
上涨家数 1000-2000 → 偏弱 → 科技有回流预期
上涨家数 2000-3000 → 平衡 → 看方向
上涨家数 > 3000  → 普涨 → 老登可能参与
上涨家数 > 4000  → 高潮 → 减仓信号

7. 证券双面判断

证券拉升 + 黄线跳水 + 老登拉升 = 背刺科技 → 减科技
证券拉升 + 黄线不跳 + 老登不动 = 助攻科技 → 持股
证券拉升 + 指数超强 + 量能够   = 可能带起科技 → 观察

8. 题材炒作节奏

第一天：新题材启动 → 果断上车
第二天：分化 → 核心票低吸
第三天：回流 → 后排减仓
第 N 天：高潮 → 全线减仓
第 N+1 天：兑现 → 等止跌
情绪票逻辑票
持有周期快进快出趋势持有
回调应对走弱即走回调是埋伏机会
选股依据涨停板、封单产业逻辑、业绩

9. 隔夜单 / 竞价技巧

  • 竞价尾声顶一字的，盘中分歧会更大
  • 关注竞价金额靠前的大票 — 它们决定科技整体方向
  • 利好：隔夜涨停排队；利空：跌停排队；超预期就撤单

10. 均线与趋势防守

5日线不破 → 持有
跌破5日线 → 减仓
跌破成本 → 先出来等止跌
5日线上方、均线乖离大 → 防回落

11. 中报/业绩炒作节奏

中报前挖坑 → 低吸机会
中报出来 + 前期没涨 → 容易修复
中报出来 + 前期大涨过 → 容易兑现
中报密集期前2周 → 是埋伏窗口
7月15日前 → 最后窗口期，还有一次高潮机会

12. 低位聊逻辑，高位看强度

低位：重点看产业逻辑、业绩预期、估值空间
高位：重点看盘口强度、资金态度、是否力竭
高位大分歧 → 聊利润垫，不聊逻辑 → 弱了崩撤溜

13. 大跌日低吸分类

不是所有深水票都能吸，按风险从低到高：

第一档（最安全）：前期跌过+快到支撑的核心票（如中际旭创、长飞光纤）
第二档（可博弈）：超跌+基本面改善+平台止跌（如通鼎互联）
第三档（有弹性）：逆势走强+被动开板（如方正科技）
第四档（不能碰）：刚破位+没止跌的前排票（如兆易创新跌停日）

三、个股案例库

9 个实战案例，按操作逻辑分类。格式：股票 → 逻辑 → 操作 → 经验

案例 1：黄河旋风

6/21–6/23 金刚石/散热 炸板六原则

逻辑：新题材第一天上车，炸板时净额流入+缩量+题材延续 → 6原则良性分歧

操作：第一天炸板低吸 → 第二天加仓 → 利润垫做厚格局 → 题材走弱边打边撤

经验：盘口强是上升期加分项，不是走弱时持有理由。题材弱了就要撤。

案例 2：金钼股份

6/21–6/30 金属/钼 反包

逻辑：波段底 + 回踩支撑有效

操作："边起立边砸，利润垫做出来" → 5分K压力位过不去就减

经验：大跌当天不抄底。反包需要量价配合。

案例 3：德明利

6/12入场–6/30新高 存储芯片 中报 7天30%

逻辑：基于中报布局，存储产业逻辑支撑

操作：6月12日附近低吸 → 利润垫50%+ → 中报前准备砸盘兑现

经验：中报前新高猛涨→出中报易兑现；中报前兑现过→出中报易修复。

案例 4：英维克

6/22一字–6/30 液冷/散热 财报分析

逻辑：液冷中军，Q1扣成本Q2变利润 → 10000%增速预期

操作："拿到出中报"，5日线不破持有

经验：财报不只数字，要看背后操作空间。

案例 5：台基股份

6/22异动–7/1 功率半导体 预期管理

逻辑：低位刚启动，"今天尾盘或明天必须启动，再不启动就真弱了"

操作：低吸 → 拉升做利润垫 → 不及预期边打边撤

经验：对个股要有明确预期和时效。队友都涨了它还不动→是弱。

案例 6：宏景科技

6/21–6/30 算力 中报

逻辑：算力涨价 + 中报逻辑

操作：埋伏半个月，上涨3-5天

经验：趋势票需要耐心等待。

案例 7：深科技

6/24–7/1 低位补涨

逻辑：涨了2倍，比其他票还属低位

操作：科技修复时下杀等修复，不封板减仓

经验：低位补涨票安全边际更高。

案例 8：北京君正

6/21首提–6/24涨停 存储补涨

逻辑：其他存储涨幅已不低时的补涨标的

操作：与深科技、太极实业同批关注

经验：热点方向内部有轮动，补涨机会值得关注。

案例 9：多氟多

6/25启动–7/1 氢氟酸 换手量

逻辑：供货台积电三星，氢氟酸+电池反推

操作：封板健康换手区间 60-80亿（前高109亿）→ 放量到100亿不健康

经验：换手量可以精算，缩量太多/放量太大都不健康。

案例 10：通鼎互联

7/2 超跌反弹 扭亏

逻辑：扭亏为盈 + 短期超跌 + 平台处止跌

操作：盘中低吸控仓。"后面下跌空间也没多少，等科技修复就容易解套"

经验：超跌+支撑+基本面改善 = 左侧博弈安全垫更厚。

案例 11：方正科技

7/2 逆势 被动开板

逻辑：科技全线大跌日逆势走强，被恐慌情绪带开板

操作：尾盘均线附近低吸。"明天若修复，它就能吃到一定溢价"

经验：逆势票+被动开板（非主动走弱）= 修复日弹性更大。

案例 12：东方锆业

7/2 国产替代 氧化锆

逻辑：5N级高纯纳米氧化锆，日本断供，国产唯一供应商

操作："不能无脑追，看趋势机会"

经验：国产替代逻辑票等趋势走出来再参与，不追消息。

经验汇总

关于买点

  1. 大跌当天不抄底
  2. 买在分歧，卖在高潮
  3. 买点后置，等换手充分
  4. 深水割肉没性价比，等反弹

关于卖点

  1. 高潮不封板 = 减仓点
  2. 三连高潮必须减
  3. 不及预期的票，边打边撤
  4. 利润垫做出来才有主动权

关于持股

  1. 利润垫厚 → 5日线防守
  2. 逻辑没坏 → 可以等
  3. 题材走弱 → 不管利润垫多少都要考虑走

四、心法与纪律

投资心态 · 交易纪律 · 成长路径 · 金句集

核心心法

关于时间
"人们总是高估 3、5 个月的努力，低估 3、5 年的坚持"
"赵老哥也才 8 年一万倍"
关于认知
"认知内的东西，用技术和逻辑去分析；认知外的东西，只能通过利润垫和仓位去防守"
关于纪律
"任何时候，都要对纪律心存敬畏"
"现在是环境好，乱搞都能挣钱。过一段时间还这么操作，一问一个不吱声"
关于成长
"交易的量不会让人成长，对每一笔交易的反思才会"

交易纪律

🟢 买入纪律

  • 不在高潮时追高
  • 不在大跌当天抄底
  • 不买杂毛，聚焦主线
  • 先做利润垫，再谈格局
  • 买点后置，等换手充分

🔴 卖出纪律

  • 高潮不封板 → 减仓
  • 三连高潮 → 必须减
  • 跌破 5 日线 → 减仓
  • 跌破成本 → 先出来等止跌
  • 题材走弱 → 边打边撤

🟡 仓位纪律

  • 不重仓单一方向
  • 有仓位，但不能满仓
  • 利润垫做出来再格局
  • 控制仓位远比选股重要

🔵 盘中纪律

  • 深水不割肉，等修复
  • 高潮不追高，等回落
  • 科技分歧不超过2天
  • 不要"一山看着一山高"

常见错误

错误后果纠正
高潮追高当天被套等分歧再参与
深水割肉割在最低点等修复再减
三连高潮不卖利润大幅回撤纪律减仓
做杂毛浪费行情聚焦主线
频繁做 T越做越亏减少无效操作
一山看着一山高追涨杀跌拿住逻辑票

成长路径

  1. 看懂盘面：知道今天在发生什么

  3. 管住手：不做杂毛，不乱追高
  4. 控制回撤：高潮减仓就是控制回撤
  5. 稳定盈利：每个月收红，复利
