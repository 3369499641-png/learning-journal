## 案例 1：黄河旋风 — 炸板六原则实战

**时间**：2025年6月21日（夏至）– 6月23日
**题材**：金刚石/散热（金刚石散热方案）
**逻辑**：
- 新题材启动第一天，果断上车
- 炸板补量时：净额流入 + 日K缩量 + 题材延续 → 6原则符合 → 良性分歧
- 第二天继续B点
- 后续盘口极强（压大单不跌 = 资金在承接）

**操作**：
- 第一天炸板低吸
- 第二天加仓
- 后续利润垫做厚后格局
- 题材走弱时边打边撤

**经验**：
- 炸板不是简单的"走弱"，要分析是出货还是补量
- 盘口强是上升期的加分项，不是走弱时的持有理由
- 题材弱了就要撤，不要因为"盘口好看"就犹豫

## 案例 2：金钼股份 — 反包与利润垫

**时间**：2025年6月21日 – 6月30日
**题材**：金属/钼
**逻辑**：
- "还在波段底"
- 回踩支撑有效后考虑
- 大跌当天不抄底，"等明天回踩支撑有效再考虑"

**操作**：
- "边起立边砸，利润垫做出来"
- 5分K压力位过不去就减仓
- "爆量不跌，5分K精准控盘，量能越来越小" → 强势

**经验**：
- 大跌当天不要抄底
- 反包需要量价配合
- 做利润垫是第一要务

## 案例 3：德明利 — 中报趋势大牛

**时间**：2025年6月12日（入场）– 6月30日（新高）
**题材**：存储芯片 / 中报业绩
**逻辑**：
- "基于中报去布局"
- 存储方向有产业逻辑支撑
- 前期大幅洗盘后走趋势

**操作**：
- 6月12日附近低吸（"不少老铁吸了德明利，收盘大哭"）
- 后续持有，利润垫50%+
- 中报前准备砸盘兑现

**经验**：
- "7个交易日，30%，要啥自行车"
- 中报前如果是新高猛涨 → 出中报容易兑现
- 中报前大幅兑现过 → 出中报容易修复
- 好票不便宜，"太贵了，阻碍了大家共同富裕"

## 案例 4：英维克 — 液冷中军与财报分析

**时间**：2025年6月22日（一字炸板）– 6月30日（玉柱福堂）
**题材**：液冷/散热
**逻辑**：
- 液冷方向板块中军
- 财报分析：Q1扣了成本，Q2货变成利润 → 业绩爆发预期
- "Q1 800万，Q2只要8个亿，就是10000%的增速"

**操作**：
- "拿到出中报"
- 5日线不破持有
- 注意量价配合

**经验**：
- 财报不只是看数字，要分析背后的操作空间
- 中军票走趋势为主，不追高
- "玉柱福堂"形态 → 慢慢往上走

## 案例 5：台基股份 — 预期管理

**时间**：2025年6月22日（首次异动）– 7月1日（最后通牒）
**题材**：功率半导体
**逻辑**：
- 低位刚启动，功率半导体方向
- "调整完需要来一根大阳线"
- "今天尾盘或者明天必须启动，如果再不启动，就真弱了"

**操作**：
- 低吸入场
- 拉升做利润垫
- 不及预期 → 边打边撤
- 主力要表态 → "不表态主力就是傻"

**经验**：
- 对个股要有明确的预期和时效
- 在这个位置，主力需要表态
- 队友都涨了它还不动 → 是弱
- "现在科技行情这么好，还在那洗盘调整，说明主力没实力"

## 经验汇总

### 关于买点
1. 大跌当天不抄底
2. 买在分歧，卖在高潮
3. 买点后置，等换手充分
4. 深水割肉没性价比，等反弹

### 关于卖点
1. 高潮不封板 = 减仓点
2. 三连高潮必须减
3. 不及预期的票，边打边撤
4. 利润垫做出来才有主动权

### 关于持股
1. 利润垫厚 → 5日线防守
2. 逻辑没坏 → 可以等
3. 题材走弱 → 不管利润垫多少都要考虑走
