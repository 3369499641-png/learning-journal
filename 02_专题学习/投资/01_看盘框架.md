# 看盘框架

## 一、盘前预案（每天必做）

### 1. 指数分析
- 前一交易日收盘位置
- 关键支撑/压力位（两道线）
- 高开/低开/平开的不同含义
- 量能预期

### 2. 情绪指标
选几个代表性标的作为风向标：
- **D头指标**（多头代表）：看封单、溢价
- **K头指标**（空头代表）：看是否修复
- 判断：D头强/D头弱 × K头强/K头弱 → 情绪偏暖/偏冷/弱平衡

### 3. 题材预判
- 竞价最强方向是什么
- 科技与老登（传统板块）的博弈预判
- 核心票竞价金额排序

### 4. 个股预案
- 持有个股的几种走势预案（强/中/弱）
- 新开仓条件

---

## 二、竞价阶段（9:15-9:25）

### 核心判断链

```
指数位置 → 情绪强弱 → 题材方向 → 个股应对
```

### 开盘三问
1. **指数开在什么位置？**（支撑之上/之下/两道线之间）
2. **涨跌家数如何？**（决定今天是抱团还是普涨）
3. **科技vs老登谁在动？**（决定资金流向）

### 关键信号
- **高开**：注意"高开分歧是卖点"
- **低开**："低开利于消化分歧，若指数修复，科技有望修复"
- **证券拉升**：判断是"助攻"还是"背刺"

---

## 三、开盘阶段（9:30-10:00）

### 运行节奏判断

科技开盘后的四种走法：

| 走法 | 应对 |
|------|------|
| 先修复 | 偏减仓点，防回落 |
| 先分歧 | 看水下低吸机会 |
| 直接走强 | 持股，不封板减 |
| 直接走弱 | 等修复，不割深水 |

### 老登与科技的博弈

```
老登拉升 + 证券背刺 → 科技减仓
老登投降 + 证券助攻 → 科技持股/加仓
老登拉升 + 量能够 → 可能高低切
老登高潮 → 科技可埋伏
```

## 四、盘中阶段（10:00-14:30）

### 10点早盘小结模板
1. **指数**：开盘位置 → 盘中走势 → 当前位置 → 今日定性
2. **情绪**：竞价情绪 → 盘中变化 → 涨跌停/涨跌比
3. **题材**：竞价方向 → 盘中轮动 → 当前主线
4. **个股**：持有个股的状态和操作建议

### 盘中关键信号
- 涨跌家数变化（最重要的实时指标）
- 黄白线关系
- 证券板块动向
- 科技核心票动向
- 新题材异动

## 五、题材运行节奏

这是复盘中反复出现的核心框架：

```
分歧 → 修复 → 高潮 → 兑现 → 分歧 → ...
```

### 分歧深度：一冰/二冰/三冰

```
第一次分歧（一冰）→ 第一时间低吸核心
第二次分歧（二冰）→ 值得博弈，偏左侧，控仓
第三次分歧（三冰）→ 概率极低，重仓机会
```

> "科技分歧一般不超过2天，第三天修复概率大"
> "二冰偏左侧，控仓；三冰大胆找机会，否极泰来"
> "科技三冰的位置还在看空，你不是我科技的兄弟，你是真空头"

### 左侧 vs 右侧

| | 左侧 | 右侧 |
|------|------|------|
| 特征 | 提前埋伏，不等确认 | 确认后再入场 |
| 优势 | 赔率高，成本低 | 确定性高 |
| 劣势 | 可能继续跌 | 成本高，利润空间小 |

### 情绪兑现 vs 逻辑兑现

大跌时必须区分：
- **情绪兑现**：逻辑没坏，市场恐慌 → 深水是机会
- **逻辑兑现**：基本面变了 → 必须离场

**案例（7月2日）**：Meta出租算力→市场担心算力过剩→科技抛压。但实际上AI Token涨价、IDC算力租赁历史高位、下游加价抢算力。→ 情绪兑现，不是逻辑兑现。

### 修复 vs 止跌

这是两个不同的概念，不能混淆：

| | 修复 | 止跌 |
|------|------|------|
| 含义 | 跌多了反弹一下 | 真正企稳，不再新低 |
| 操作 | 高点减仓做利润垫 | 可以开始埋伏 |
| 风险 | 可能只是下跌中继 | 确定性更高 |

> "今天是修复，不是止跌。昨天入场的，今天给了高点，要减仓做利润垫"
> "修复不是一蹴而就的，今天再洗洗空头，后面就好一点了"

### 节奏总结：一冰→二冰→三冰的完整周期

```
一冰（高潮次日）→ 疯狂减仓，大跌当天不抄底
二冰           → 偏左侧博弈，控仓
三冰           → 否极泰来，大胆找机会
修复           → 高点做利润垫，不是加仓点
止跌           → 等确认后埋伏
```

### 各阶段操作
| 阶段 | 操作 |
|------|------|
| 分歧（第一天） | 核心票低吸，不追后排 |
| 修复 | 持股，不封板减 |
| 高潮 | **纪律减仓**，不追高 |
| 三连高潮 | 必须减仓，风险>机会 |
| 兑现/大分歧 | 深水不割，等修复再减 |

### 判断口诀
- "高潮一定是减仓点"
- "分歧是低吸核心的机会"
- "科技分歧一般不超过2天，第三天修复概率大"
- "连续高潮次日冲高容易回落"

## 七、看盘进阶：从数据到判断

### 信息处理的层次
1. **看到**：指数点位、涨跌家数、板块涨跌
2. **判断**：为什么涨/跌？谁在动？
3. **应对**：基于判断的操作策略
4. **复盘**：判断对不对？为什么？

### 经典判断链示例
```
涨跌家数4000家下跌 + 老登没起来 + 量能够 
→ 资金没地方去 
→ 只能回流科技 
→ 科技深水是低吸机会
```
