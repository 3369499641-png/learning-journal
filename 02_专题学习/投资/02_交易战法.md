# 交易战法

## 一、炸板六原则（6原则）

用于判断炸板后的回封概率：

```
1. 炸板时净额是否流入（不是流出）
2. 日K是否缩量（不是爆量）
3. 题材是否还在延续
4. 换手是否充分
5. 板块是否有支撑
6. 指数是否配合
```

**经典案例**：黄河旋风 — 炸板时净额流入、日K缩量、题材延续 → 良性分歧 → 回封

### 炸板补量
- 一字板竞价尾声顶一字的 → 盘中分歧更大 → 等换手
- "换手到前高70%以上（经验值）" → 安全区间
- "要么要确定性，要么要日内赔率"

---

## 二、789减仓法

高潮不封板的纪律性减仓：

```
科技高潮时，手里票 7%、8%、9% 又不封板 → 减仓
因为高潮都不封板，板块回落时它回落的概率大
```

**20cm票**：double一下，14%-18%不封板就减

---

## 三、买点后置

不着急在开盘追，等换手充分：

```
买点后置 = 等换手到前高70%以上
→ 确定性买点（回封确认后）
→ 日内赔率买点（深水低吸）
```

---

## 四、利润垫策略

### 核心理念
- 买入后第一时间做利润垫
- 利润垫 = 减仓收回成本，留利润博弈
- "利润垫够了就躺好"

### 操作步骤
1. 买入后，拉升减仓 → 收回部分成本
2. 剩余仓位全是利润 → 心态稳
3. 5日线不破就拿着
4. 新高附近减仓做倒T

### 不同利润垫的应对
| 利润垫厚度 | 策略 |
|-----------|------|
| <10% | 谨慎，随时准备减 |
| 10-30% | 5日线防守 |
| 30-50% | 可以格局，等新高 |
| >50% | 躺平，新高减即可 |

---

## 五、高位看金额，低位看量

```
高位：只有绝对金额才能承接得住
低位：看量能变化，放量+止跌 = 底部信号
```

### 爆量模型
- 爆量不跌 = 可能是吃货量
- 爆量+反包 = 趋势预期
- 爆量+下跌 = 出货

---

## 六、涨跌家数判断法

这是最重要的盘中指标之一：

```
上涨家数 < 1000 → 极度冰点 → 资金抱团科技/主线
上涨家数 1000-2000 → 偏弱 → 科技有回流预期
上涨家数 2000-3000 → 平衡 → 看方向
上涨家数 > 3000 → 普涨 → 老登可能参与
上涨家数 > 4000 → 高潮 → 减仓信号
```

---

## 七、证券双面判断

```
证券拉升 + 黄线跳水 + 老登拉升 = 背刺科技 → 减科技
证券拉升 + 黄线不跳 + 老登不动 = 助攻科技 → 可以持股
证券拉升 + 指数超强 + 量能够 = 可能带起科技 → 观察
```

---

## 八、题材炒作节奏

### 炒作阶段

```
第一天：新题材启动 → 果断上车
第二天：分化 → 核心票低吸
第三天：回流 → 后排减仓
第N天：高潮 → 全线减仓
第N+1天：兑现 → 等止跌
```

### 情绪票 vs 逻辑票

| | 情绪票 | 逻辑票 |
|------|------|------|
| 持有周期 | 快进快出 | 趋势持有 |
| 回调应对 | 走弱即走 | 回调是埋伏机会 |
| 选股依据 | 涨停板、封单 | 产业逻辑、业绩 |

---

## 九、隔夜单/竞价技巧

### 竞价尾声判断
- "竞价尾声顶一字的，盘中分歧会更大"
- 关注竞价金额靠前的大票 — 它们决定科技整体方向

### 隔夜单操作
- 利好：隔夜涨停排队
- 利空：跌停排队
- 超预期就撤单

## 十、均线与趋势防守

```
5日线不破 → 持有
跌破5日线 → 减仓
跌破成本 → 先出来等止跌
5日线上方、均线乖离大 → 防回落
```

---

## 十一、中报/业绩炒作节奏

```
中报前挖坑 → 低吸机会
中报出来 + 前期没涨 → 容易修复
中报出来 + 前期大涨过 → 容易兑现
7月15日前 → 最后窗口期，还有一次高潮机会
```

---

## 十二、低位聊逻辑，高位看强度

```
低位：重点看产业逻辑、业绩预期、估值空间
高位：重点看盘口强度、资金态度、是否力竭
高位大分歧 → 聊利润垫，不聊逻辑 → 弱了崩撤溜
```

---

## 十三、大跌日低吸四档分类

不是所有深水票都能吸，按风险从低到高：

```
第一档（最安全）：前期跌过+快到支撑的核心票
  例：中际旭创、长飞光纤（7月2日）

第二档（可博弈）：超跌+基本面改善+平台止跌
  例：通鼎互联（7月2日）

第三档（有弹性）：逆势走强+被动开板
  例：方正科技（7月2日）

第四档（不能碰）：刚破位+没止跌的前排票
  例：兆易创新跌停日
```

---

## 十四、修复 ≠ 止跌

```
修复：跌多了反弹一下 → 高点减仓做利润垫，不是加仓点
止跌：真正企稳不再新低 → 可以开始埋伏

修复不是一蹴而就的，可能还要洗几次空头
```

---

## 十五、只吃短线最肥美的一段

```
核心原则：只吃短线最肥美的一段，不参与调整

参与调整的大多数情况 = 倒在拉升的黎明前

解决办法：
1. 有利润垫 → 放着等
2. 没有利润垫 → 不要买在调整期
```

---

## 十六、三冰节点

```
一冰（高潮次日）→ 疯狂减仓，大跌当天不抄底
二冰            → 偏左侧博弈，控仓
三冰            → 否极泰来，大胆找机会
修复            → 高点做利润垫，不是加仓点
止跌            → 等确认后埋伏
```
