# 投资心法与纪律

## 核心心法

### 关于时间
> "人们总是高估3、5个月的努力，低估3、5年的坚持"
> "赵老哥也才8年一万倍"

### 关于认知
> "认知内的东西，用技术和逻辑去分析；认知外的东西，只能通过利润垫和仓位去防守"

### 关于纪律
> "任何时候，都要对纪律心存敬畏"
> "现在是环境好，乱搞都能挣钱。过一段时间还这么操作，一问一个不吱声"

### 关于成长
> "交易的量不会让人成长，对每一笔交易的反思才会"
> "每个人都有自己的节奏，比之前的自己进步一点也是好的"

### 关于心态
> "挣钱的时候不要怕，退潮的时候要敏"
> "看准一个方向，做出利润垫了，就等新高"

## 交易纪律

### 买入纪律
- [ ] 不在高潮时追高
- [ ] 不在大跌当天抄底（等第二天确认）
- [ ] 不买杂毛，聚焦主线
- [ ] 先做利润垫，再谈格局
- [ ] 买点后置，等换手充分

### 卖出纪律
- [ ] 高潮不封板 → 减仓
- [ ] 三连高潮 → 必须减
- [ ] 跌破5日线 → 减仓
- [ ] 跌破成本 → 先出来等止跌
- [ ] 题材走弱 → 边打边撤
- [ ] 利润垫不够 → 减仓防守

### 仓位纪律
- [ ] 不重仓单一方向
- [ ] 有仓位，但不能满仓
- [ ] 利润垫做出来再格局
- [ ] "控制仓位远比选股重要"

### 盘中纪律
- [ ] 深水不割肉，等修复
- [ ] 高潮不追高，等回落
- [ ] 科技分歧不超过2天，第三天修复概率大
- [ ] 不要"一山看着一山高"
- [ ] 分时低点+分时高点 → 一个买点一个卖点 → 最丝滑

## 常见错误

| 错误 | 后果 | 纠正 |
|------|------|------|
| 高潮追高 | 当天被套 | 等分歧再参与 |
| 深水割肉 | 割在最低点 | 等修复再减 |
| 三连高潮不卖 | 利润大幅回撤 | 纪律减仓 |
| 做杂毛 | 浪费行情 | 聚焦主线 |
| 频繁做T | 越做越亏 | 减少无效操作 |
| 一山看着一山高 | 追涨杀跌 | 拿住逻辑票 |

## 关于成长路径

### 新手 → 老手的阶段
1. **看懂盘面**：知道今天在发生什么
2. **用对战法**：选对适合自己的1-2个战法做精
3. **管住手**：不做杂毛，不乱追高
4. **控制回撤**：高潮减仓就是控制回撤
5. **稳定盈利**：每个月收红，复利

### 太阳兄弟的感悟（群友总结）
> "市场情绪炒作这种风格在慢慢演变...更多的是看产业逻辑，看个股基本面，就是大局观。首先大局观必须得对题材有足够的预期，才能知道题材的持续性，以这个作为基础，再结合战法，而且战法不需要用那么多，只需要用自己用着最舒服的那一种，做精就行。还有就是行情好的时候，千万不要觉得自己行了，要敬畏市场。"
