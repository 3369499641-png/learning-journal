# 学习总结模仿进步

> 长线项目 — 每天积累一点点，持续迭代。

## 目录结构

| 文件夹 | 用途 |
|--------|------|
| `01_每日记录/` | 每天的学习笔记、收获、灵感 |
| `02_专题学习/` | 按主题组织的深度学习内容 |
| `03_模仿练习/` | 仿写、仿做、代码复现等实践 |
| `04_复盘总结/` | 周/月/季度回顾与反思 |
| `05_参考资料/` | 收藏的文章、代码、资源归档 |
| `06_进度追踪/` | 目标设定、进度看板、里程碑 |

## 使用方式

### 每日流程
1. **学** → 记录到 `01_每日记录/`，按日期归档
2. **练** → 模仿练习放到 `03_模仿练习/`
3. **复盘** → 周末回顾写总结到 `04_复盘总结/`

### 文件命名规范
- 每日记录：`YYYY-MM-DD_主题.md`
- 专题文章：`01_专题名_副标题.md`
- 复盘总结：`YYYY-Www_周总结.md` 或 `YYYY-MM_月总结.md`

### 标签系统
在每篇笔记头部用标签标记：
- `#概念` `#技巧` `#工具` `#思维` `#项目`
- `#待深挖` `#已掌握` `#练习中`
